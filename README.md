# 🕒 Gerçek Zamanlı Saat

HTML, CSS ve JavaScript kullanılarak geliştirilmiş gerçek zamanlı saat uygulamasıdır.

## 📌 Hakkında
Bu proje, JavaScript ile gerçek zamanlı veri güncellemelerini ve DOM manipülasyonunu öğrenmek amacıyla geliştirilmiştir. Uygulama, sistem saatini kullanarak saat, dakika ve saniye bilgilerini anlık olarak ekranda göstermektedir.

## ✨ Özellikler
- Gerçek zamanlı saat gösterimi
- Saat, dakika ve saniye görüntüleme
- Otomatik güncelleme
- Basit ve kullanıcı dostu arayüz
- Responsive tasarım

## 🛠️ Kullanılan Teknolojiler
- HTML5
- CSS
- JavaScript
- Visual Studio Code (VS Code)
